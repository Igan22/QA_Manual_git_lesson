Налаштування та ініціалізація
git config --global user.name "Ваше Ім'я": Встановлює ім'я автора для всіх локальних коммітів.
git config --global user.email "your@email.com": Задає електронну пошту автора.
git init: Створює новий порожній Git-репозиторій у поточній папці.
git clone <url>: Копіює вже існуючий віддалений репозиторій на ваш комп'ютер.

Робота зі змінами (Локально)
git status: Показує стан файлів: змінені, видалені чи невідстежувані.
git add <назва_файлу>: Додає конкретний файл до індексу для наступного комміту.
git add .: Додає абсолютно всі змінені та нові файли у поточній папці.
git commit -m "Ваше повідомлення": Фіксує збережені зміни в історії репозиторію.
git diff: Демонструє різницю між поточними файлами та останнім коммітом
